4-digit Position Match Game

Game Rules

  • Two players take turns to guess each other's secret 4-digit number.
  • If no other player is available, you can play alone against the computer.
  • When you guess, the opponent tells you which digit position(s) match.
  • Example: Secret = 4678, Guess = 2379, Response: "Ten's position matches".
  • Players alternate turns guessing until someone correctly guesses the full number.
